# Ejercicios de Programación: Java a Python 🐍

Este repositorio contiene una colección de ejercicios de programación originalmente implementados en Java, ahora portados a **Python**. El objetivo principal es practicar la sintaxis de Python manteniendo la lógica algorítmica desarrollada previamente.

## 📌 Descripción del Proyecto
Este proyecto sirve como un registro de aprendizaje y transición entre lenguajes. Cada ejercicio ha sido refactorizado para seguir las convenciones de Python (PEP 8), aprovechando sus estructuras de datos nativas (listas, diccionarios, sets) y su sintaxis concisa.
